## Project Overview

**ProgressApp** is a web application designed to help users:
- Set goals across different aspects of life (health, career, personal development, etc.)
- Receive personalized tasks and milestones to achieve those goals
- Track progress and visualize improvements over time

The app provides motivation and structure by breaking down larger life goals into actionable, trackable tasks.

## Git Workflow

**Important:** After every change to the codebase, you must:
1. Commit the changes to the local git repository with a descriptive commit message
2. Push the changes to GitHub

This ensures no progress is lost and maintains a complete history of all changes.

Example:
```
git add .
git commit -m "Add user authentication system"
git push origin main
```
